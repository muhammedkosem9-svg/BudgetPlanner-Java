SmartBudget: Personal Finance Assistant
A console-based Java application designed to help users track their daily income and expenses.
This project was developed as a practice for fundamental programming concepts like Loops, Switch-Case statements, and Logic Building.

Features
Income Management: Add your daily or monthly income.

Expense Tracking: Categorize your spending (Food, Transport, etc.).

Financial History: View a summary of your transactions and current balance.

Safe Spending: Built-in logic to prevent spending more than your current balance.

Technologies Used
Java SE (Standard Edition)
